# Frame Architect

Frame Architect is the second built-in guided campaign. It focuses on the structure of the accessibility relation rather than valuations.

The seven missions cover:

1. Reflexive Repair
2. Serial Without Reflexivity
3. Symmetric but Not Transitive
4. Transitive and Serial, Not Reflexive
5. Euclidean but Not Transitive
6. S4-Style Frame
7. Final Architecture

Missions use the property convention already implemented by the frame evaluator. In particular, Euclideanity is checked as: if `xRy` and `xRz`, then `yRz`.

The last mission starts from a reflexive, transitive, non-symmetric preorder and asks the player to transform it into a non-trivial equivalence relation. It records the resulting construction through the shared campaign progress system.
